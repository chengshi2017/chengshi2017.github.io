jquery+ajax实现数据的批量删除

发表于 2018-03-28 | 分类于 前端
阅读时长 ≈ 7

  在写项目的时候，由于数据比较多，删除数据的时候一个一个的点击删除比较麻烦，所以准备在项目中使用批量删除的功能。批量删除在项目中的实现比较简单。

批量删除功能的实现主要难度是在前端，在后端只需要用mybatis做一个批量删除的接口就行。
