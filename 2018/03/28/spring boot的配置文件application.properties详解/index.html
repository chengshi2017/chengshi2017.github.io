spring boot的配置文件application.properties详解

发表于 2018-03-28 | 分类于 后端
阅读时长 ≈ 14

  spring boot 设计之初的目的就是用来简化spring 应用的初始化搭建以及开发方式，使我们不用遵循固定的样本式配置，简化了开发过程。

但是我们在开发的过程中，却有些用法不得不配置，spring boot也给出了解决的办法，那就是配置全部都写在.properties或者.yml文件中。
此博客正是记录.properties文件中的一些常用配置。
